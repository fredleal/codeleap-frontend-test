# CodeLeap Frontend Test

A social network feed app built as a frontend coding test for CodeLeap.

**Live demo:** https://codeleap-frontend-test-h4ucfbgmi-fredleals-projects.vercel.app

## Stack

- React 19 + TypeScript
- Vite
- TanStack React Query
- Tailwind CSS 4
- React Router

## Running locally

```bash
bun install
bun run dev
```

## What was built

### Core

- Signup screen with username
- Full CRUD on posts (create, read, edit, delete)
- Integration with CodeLeap API
- Infinite scroll / pagination

### Bonus

- Like/unlike posts
- Login/logout flow
- Fully responsive layout
- Hover effects and smooth transitions
